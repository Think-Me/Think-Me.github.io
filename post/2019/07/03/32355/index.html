dockerе’Ңdocker-composeзҡ„е®үиЈ…

еҸ‘иЎЁдәҺ2019-07-03|жӣҙж–°дәҺ2019-07-03|е®үе…ЁдёҺиҝҗз»ҙ
|еӯ—ж•°жҖ»и®Ў:936|йҳ…иҜ»ж—¶й•ҝ:3еҲҶй’ҹ

е…Ҳд»Ӣз»ҚдёҖдёӢdockerжҳҜе•Ҙпјҡ

Docker жҳҜдёҖдёӘејҖжәҗзҡ„еә”з”Ёе®№еҷЁеј•ж“ҺпјҢи®©ејҖеҸ‘иҖ…еҸҜд»Ҙжү“еҢ…д»–д»¬зҡ„еә”з”Ёд»ҘеҸҠдҫқиө–еҢ…еҲ°дёҖдёӘеҸҜз§»жӨҚзҡ„е®№еҷЁдёӯ,з„¶еҗҺеҸ‘еёғеҲ°д»»дҪ•жөҒиЎҢзҡ„LinuxжңәеҷЁжҲ–Windows жңәеҷЁдёҠ,д№ҹеҸҜд»Ҙе®һзҺ°иҷҡжӢҹеҢ–,е®№еҷЁжҳҜе®Ңе…ЁдҪҝз”ЁжІҷз®ұжңәеҲ¶,зӣёдә’д№Ӣй—ҙдёҚдјҡжңүд»»дҪ•жҺҘеҸЈгҖӮ

еҶҚиҜҙдёҖдёӢdocker composeжҳҜе•Ҙпјҡ

docker composeжҳҜdockerдёүеү‘е®ўд№ӢдёҖпјҢз”ЁдәҺе®ҡд№үе’ҢиҝҗиЎҢеӨҡдёӘе®№еҷЁ Docker еә”з”ЁзЁӢеәҸзҡ„е·Ҙе…·гҖӮйҖҡиҝҮdocker composeпјҢдҪ еҸҜд»ҘдҪҝз”ЁYMLж–Үд»¶жқҘй…ҚзҪ®еә”з”ЁзЁӢеәҸйңҖиҰҒзҡ„жүҖжңүжңҚеҠЎгҖӮз„¶еҗҺпјҢдҪҝз”ЁдёҖдёӘе‘Ҫд»ӨпјҢе°ұеҸҜд»Ҙд»ҺYMLж–Үд»¶й…ҚзҪ®дёӯеҲӣе»әе№¶еҗҜеҠЁжүҖжңүжңҚеҠЎгҖӮComposeеҜ№еә”зҡ„е‘Ҫд»Өдёәdocker-composeгҖӮ
з®ҖеҚ•дёҖзӮ№иҜҙе°ұжҳҜжҠҠдёҖе Ҷdockerз»„еҗҲзј–жҺ’еҲ°дёҖиө·д»Ҙе®һзҺ°еҝ«йҖҹз®ЎзҗҶгҖӮ

жң¬ж–Үзҡ„зӣ®зҡ„жҳҜдёәдәҶж–№дҫҝжҲ‘й…ҚзҪ®swarmпјҢжүҖд»Ҙз”Ёзҡ„зүҲжң¬йғҪжҳҜжҜ”иҫғж–°зҡ„гҖӮеҰӮжһңжӮЁеҸӘжҳҜеҚ•зәҜзҡ„дҪҝз”ЁdockerжҲ–иҖ…docker composeеҲҷж— йңҖзү№еҲ«ж–°зҡ„зүҲжң¬

еҘҪдәҶпјҢеӣһеҪ’жӯЈйўҳ

1гҖҒеҰӮдҪ•е®үиЈ…docker

е…ҲжҹҘзңӢеҶ…ж ёзүҲжң¬д»ҘзЎ®е®ҡе…је®№е“ӘдәӣdockerзүҲжң¬

еҰӮжһңиҝҷдёӘең°ж–№жҳҫзӨәеҶ…ж ёзүҲжң¬дёә3.10.0еҲҷжңҖеӨ§ж”ҜжҢҒеҲ°docker-18.03.1.ceзүҲжң¬
жҲ‘д»¬жғіе®үиЈ…жҜ”иҫғж–°зҡ„зүҲжң¬гҖҗеӣ дёәж—§зүҲжң¬дёҚж”ҜжҢҒжңүдәӣж–°зү№жҖ§гҖ‘пјҢжүҖд»ҘжҲ‘д»¬йҰ–е…ҲиҰҒеҚҮзә§жңҚеҠЎеҷЁзҡ„еҶ…ж ёзүҲжң¬

жңҚеҠЎеҷЁжӯЈеёёйҮҚеҗҜд№ӢеҗҺеҸҜд»ҘзңӢдҪ еҝғжғ…йҖүжӢ©жҳҜеҗҰжү§иЎҢд»ҘдёӢе‘Ҫд»ӨеҺ»еҲ йҷӨж—§еҶ…ж ёгҖҗеҸҜйҖүйЎ№гҖ‘

####ok,зҺ°еңЁеҸҜд»ҘејҖе§Ӣе®үиЈ…dockerдәҶ
е…ҲзңӢзңӢжҳҜеҗҰжңүж—§зүҲжң¬зҡ„dockerпјҢеҰӮжһңжңүеҸҜд»Ҙз”Ёд»ҘдёӢе‘Ҫд»ӨеҺ»еҚёиҪҪ

  • йҰ–е…ҲжҲ‘д»¬иҰҒжӣҙж–°дёҖдёӢyumеҢ…
  • е…¶ж¬ЎиҰҒе®үиЈ…иҰҒдҫқиө–зҡ„иҪҜд»¶еҢ…гҖҗyum-util жҸҗдҫӣ yum-config-manager еҠҹиғҪпјҢ еҸҰеӨ–дёӨдёӘжҳҜ devicemapper й©ұеҠЁдҫқиө–гҖ‘
  • з„¶еҗҺи®ҫзҪ®yumжәҗ
  • еҶҚжҹҘзңӢд»“еә“дёӯжүҖжңүdockerзүҲжң¬пјҢйҖүжӢ©дҪ жғіиҰҒзҡ„зүҲжң¬еҺ»е®үиЈ…
  • иҝҷйҮҢжҲ‘йңҖиҰҒdocker-18.06.3.ceзүҲжң¬пјҢдҪ д»¬жҢүз…§дҪ д»¬зҡ„йңҖжұӮеҺ»йҖүжӢ©зүҲжң¬
  • жңҖеҗҺеҗҜеҠЁе№¶еҠ е…ҘејҖжңәиҮӘеҗҜ

2гҖҒеҰӮдҪ•е®үиЈ…docker compose

йҖҡиҝҮи®ҝй—® https://github.com/docker/compose/releases еҸҜд»ҘжҹҘзңӢзҺ°жңүзҡ„жүҖжңүзүҲжң¬пјҢеҺ»йҖүжӢ©еҗҲйҖӮзҡ„зүҲжң¬гҖӮ
иҝҷйҮҢз”ұдәҺжҲ‘йңҖиҰҒжңҖж–°зҡ„3.7зҡ„зү№жҖ§пјҢжүҖд»ҘеҲҡжүҚdockerе®үиЈ…зҡ„зүҲжң¬жҳҜ18.06.3пјҢdocker-composeжҲ‘д№ҹйҖүжӢ©жңҖж–°зҡ„гҖҗ1.24.1зүҲжң¬гҖ‘еҺ»е®үиЈ…д»Ҙж”ҜжҢҒж–°зү№жҖ§гҖӮ

жңҖз»ҲжҳҫзӨәdocker-composeзҡ„зүҲжң¬еҚідёәе®үиЈ…жҲҗеҠҹгҖӮ

жіЁж„Ҹпјҡ dockerе’Ңdocker composeй…ҚзҪ®ж–Үд»¶йҮҢйқўзүҲжң¬иҰҒжҢүз…§дёӢйқўиЎЁж јиҰҒжұӮеҜ№еә”
